# Segment Tree ITMO Solutions

A collection of my solutions to the **ITMO Academy Segment Tree Course** and related competitive programming problems solved in **C++**.

---

## About

This repository contains implementations and solutions for various Segment Tree concepts including:

- Point Updates
- Range Queries
- Lazy Propagation
- Range Assignment
- Maximum Subarray Sum
- Nested Segments
- Inversion Counting
- Bitwise Operations on Segments
- Advanced Segment Tree Techniques

The goal of this repository is to improve problem-solving skills and maintain clean implementations of important data structures used in competitive programming.
